*For flask-project

First
1. download file in this link "https://drive.google.com/file/d/1I91uLQatZcLRk7CwHZN9V9BH8hDXBpJN/view?usp=sharing".
2. unzip downloaded file and put ml-model folder into flask-project folder.

type command
1. cd flask-project
2. pip install -r requirements.txt
3. flask run --port 2000

swagger ui at: http://127.0.0.1:2000

*For client

1. cd client
2. pip install -r requirements.txt
3. flask run

web app at: http://127.0.0.1:5000
